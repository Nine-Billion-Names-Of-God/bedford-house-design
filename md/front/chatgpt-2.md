| Item                 | Requirement / constraint                                                                                                            |
| -------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Location and style   | Victorian terraced house in Bedford Park, Chiswick; planting should feel appropriate to a traditional London front garden           |
| Orientation          | West-facing; shade in morning, partial sun midday, stronger sun later in day                                                        |
| Soil                 | Typical London clay assumed; plants should tolerate heavier soil once improved with organic matter                                  |
| Layout               | **Must stay exactly as planned**; no moving path, gate, shed, wall, fence, hedges, house features, or tap                           |

| Privacy              | Front boundary planting should give **soft evergreen screening**, but not read as a clipped hedge                                   |
| Height limits        | Most planting under **1.5 m**; tallest elements concentrated near boundary, up to about **2.0 m max**                               |
| Character            | Structured but natural; tidy and organised, not formal, not sprawling cottage garden                                                |
| Maintenance          | Moderate maintenance acceptable                                                                                                     |
| Central focal plant  | May be added only if compact, visually strong, and does **not** obstruct bike circulation                                           |

## 2. Planting structure for the whole garden

| Layer                        | Role in the design                                         | Where it should be strongest                                              |
| ---------------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------- |
| Evergreen framework          | Gives winter presence, privacy, and structure              | Front boundary bed, parts of left hedge strip, selected house wall shrubs |
| Medium flowering shrubs      | Main seasonal volume and character                         | Front boundary bed and left hedge strip                                   |
| Perennials and lower fillers | Long flowering season, softens edges, ties shrubs together | All planted zones                                                         |
| Climbers                     | Soften brickwork and connect garden to house               | House wall bed near door / around bay where suitable                      |
| Ground layer / edging        | Keeps the design neat, covers soil, helps suppress weeds   | Front edge of beds, house wall bed, hedge strip                           |
| Optional focal specimen      | Creates a controlled moment in the shingle court           | Central court only if positioned off bike turning line                    |

**Overall approach:**
The garden should read as a planted frame around an open centre. The strongest evergreen mass should sit along the **front boundary**, because that is where privacy is needed. The **left hedge strip** should be treated as a slim layered border to enrich the existing hedge base. The **house wall bed** should be lighter and more controlled, with climbers and narrower plants so the façade is softened without blocking the tap or circulation. The **central shingle court** remains visually open and practical.

## 3. Zone-by-zone planting recommendations

### Zone 1 — Left Hedge Strip

| Aspect           | Recommendation                                                                                                 |
| ---------------- | -------------------------------------------------------------------------------------------------------------- |
| Purpose          | Enrich the base of the hedge and stop it feeling like a plain side strip                                       |
| Character        | Layered but narrow planting; tidy base with seasonal highlights                                                |
| Best plant types | Shade/part-sun tolerant evergreen shrubs, compact flowering shrubs, perennials that cope with root competition |
| Height           | Mostly 40–120 cm, with a few taller shrubs up to 150 cm if kept narrow                                         |
| Notes            | Avoid bulky plants spilling into the path                                                                      |

### Zone 2 — House Wall Bed

| Aspect           | Recommendation                                                              |
| ---------------- | --------------------------------------------------------------------------- |
| Purpose          | Soften house wall and bay window, link garden to building                   |
| Character        | More refined and narrower than boundary bed                                 |
| Best plant types | Climbers on trellis/wires, compact shrubs, neat perennials, aromatic edging |
| Height           | Mostly 30–120 cm, climbers higher on wall                                   |
| Notes            | Keep clear access to the tap; avoid dense shrubs directly in front of it    |

### Zone 3 — Front Boundary Bed

| Aspect           | Recommendation                                                                                |
| ---------------- | --------------------------------------------------------------------------------------------- |
| Purpose          | Main privacy layer and visual planting statement                                              |
| Character        | Layered, airy screening with evergreen backbone and long-season flowers                       |
| Best plant types | Upright evergreen shrubs, hydrangeas or similar flowering shrubs, grasses/perennials at front |
| Height           | Front edge 20–60 cm, middle 60–120 cm, back layer 150–200 cm max                              |
| Notes            | Should screen softly, not form a continuous hedge wall                                        |

### Zone 4 — Central Shingle Court

| Aspect           | Recommendation                                                                             |
| ---------------- | ------------------------------------------------------------------------------------------ |
| Purpose          | Bike manoeuvring, circulation, openness                                                    |
| Character        | Mostly open; any planting must be minimal and deliberate                                   |
| Best plant types | Optional compact focal specimen only                                                       |
| Height           | Usually 100–180 cm depending on specimen                                                   |
| Notes            | Keep the main turning and access lines open, especially between path, shed, and front door |

## 14. Final recommended planting scheme

### Recommended scheme: **Scheme A — Classic Bedford Park soft-structured scheme**

| Why it is the best fit             | Explanation                                                                         |
| ---------------------------------- | ----------------------------------------------------------------------------------- |
| Best match to Bedford Park context | It feels traditional, settled, and appropriate to a Victorian front garden          |
| Balanced privacy                   | It screens the boundary without turning into a hedge wall                           |
| Good all-year performance          | Strong evergreen bones plus dependable summer flowering                             |
| Works with the layout              | It respects the open shingle centre and does not overload the space                 |
| Manageable maintenance             | It needs care, but not constant intervention                                        |
| Good visual hierarchy              | Tallest and strongest planting sits where it should: boundary and edges, not centre |

### My refinement to Scheme A

I would use:

* **Taxus + Osmanthus + compact hydrangeas** as the main boundary composition
* **Skimmia + hellebores + geraniums** in the left hedge strip
* **Trachelospermum + Choisya + heuchera/erigeron** on the house wall
* **Syringa meyeri ‘Palibin’** only if, after setting out, the turning circle still feels generous

### If you want the safest central focal option

Choose **Syringa meyeri ‘Palibin’**.

### If you want no risk to circulation at all

Leave the central court open and let the boundary planting provide the focal effect.
